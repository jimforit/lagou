岗位职责：

负责 Web 或小程序的开发，高效优质完成产品界面和功能的实现。持续优化 Web 前端体验和性能，改进现有的技术框架，提升前端开发效率。


任职要求：

1. 三年以上前端工作经验。

2. 精通 HTML/CSS，关注前端用户体验。

2. 熟悉 ES2015 以及模块化开发，熟悉 Node，熟悉 Vue/React 等前端框架。

3. 熟悉前端工程化以及 Webpack/Gulp 等构建工具。

4. 熟悉 HTTP 协议以及前端性能优化，关注前端技术发展。

5. 具备用户体验意识，优秀的代码风格，良好的文档编写能力。


优先条件：

1. 开发或维护过开源项目

2. 提供 GitHub 帐号、StackOverflow 帐号、知乎帐号、技术 Blog 链接等
